Creating an EGL package

You can create an EGL package. An EGL package is a named collection of related source parts.

Before you can perform this task you must create an EGL project or a folder to contain the package to be created.
To create an EGL package, do as follows:
  1. Select a project or folder to contain the EGL package.
  2. From the main menu, click File > New > Other. The New window opens.
  3. In the New window, select EGL > Package, and click Next. The New EGL Package window is displayed.
  4. In the New EGL Package window, do as follows:
    1. If the Source folder field is not prepopulated, specify the source folder.
    2. In the Package name field, type the name of the EGL package to be created.
    3. Click the Finish button.
