Creating an EGL source folder

After you create a project in the workbench, you can create one or more folders within that project to contain your EGL files.

Before you can perform this task you must create an EGL project.
To create a folder for grouping EGL files, do as follows:
  1. From the main menu, click File > New > Other. The New window opens.
  2. In the New window, select EGL > Source Folder, and click Next. The New Source Folder window is displayed.
  3. In the New Source Folder window, do as follows:
    1. In the Project name field, specify the name of the project to contain the new source folder.
    2. In the Folder name field, specify a name for the folder to be created.
    3. Click the Finish button.
