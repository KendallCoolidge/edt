Importing EGL deployment descriptors

You can specify the deployment descriptors that are referenced from the deployment descriptor being defined. The content in those files is made available when you generate or deploy the deployment descriptor.

  • To add a deployment descriptor, click the Add button, expand folders as appropriate, and double-click a file that has the extension .egldd.
  • To remove a deployment descriptor, select it in the grid and click the Remove button.
  • To open a deployment descriptor in the EGL deployment descriptor editor, select it in the grid and click the Open button.
