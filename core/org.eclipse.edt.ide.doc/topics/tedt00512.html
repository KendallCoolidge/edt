Setting EGL formatter preferences

You can set preferences for formatting in the EGL editor.

Do as follows:
  1. From the main menu, click Window > Preferences.
  2. From the navigation tree, expand EGL and click Editor > Formatter.
  3. In the Formatter window, you can add, modify, rename, remove, export, or import editor formatting profiles.
    Task Action
    Create a new profile Click New and select the features for the profile on the Edit Profile window.
    Change the settings defined in an existing profile Select the name of the profile from the drop-down list, click Edit, edit the settings, and click OK.
    Change the name of a profile Select the profile, click Rename, type the new name, and click OK.
    Remove a profile Select the profile, click Remove, and click Yes.
    Export a profile Select the profile, click Export, specify the location and name of the file that you want to save, and click Save.
    Import a profile Click Import, select the XML file that you want to import, click Open, and click OK.
  4. To save the changes and remain in the Preferences window, click Apply. To save the changes and exit the window, click OK.
