Setting preferences for EGL Rich UI deployment

The deployment preferences affect the Rich UI deployment tab of the EGL Deployment Descriptor editor.

To set the initial Rich UI preferences, do as follows:
  1. From the main menu, click Window > Preferences. The Preferences window is displayed.
  2. Expand EGL and Rich UI and then click Deployment. The Deployment pane is displayed.
  3. In the Locales area, you select which locales are supported by default when you open the Rich UI Deployment tab in the EGL Deployment Descriptor editor. For details on the use of locales, see "Use of properties files for displayable text." All the locales that are available to you in the Rich UI Application locales area are listed in the Rich UI pane, as described in "Setting preferences for Rich UI."
  4. If you want to return the settings on the Deployment pane to the original product settings, click Restore Defaults.
  5. To save the changes and remain in the Preferences window, click Apply. To save the changes and exit the window, click OK. To cancel the changes and exit the window, click Cancel.
