Updating resources in the EGL deployment descriptor

You can select files or groups of files that are to be included in or omitted from a deployment. By default, files in the project itself are accessible, as are files in every project in the EGL build path. The deployable files are in JavaSource and WebContent folders.

  1. Open the EGL Deployment Descriptor editor by double-clicking an EGL deployment descriptor.
  2. Click Resources. The Resources tab is displayed.
  3. To select files or groups of files that are to be made available to the user, select check boxes. To deselect files or groups of files that are not to be made available to the user, clear check boxes. Expand or contract projects, folders, and packages as needed to access file entries.
