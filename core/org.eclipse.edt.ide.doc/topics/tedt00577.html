Setting EGL SQL preferences

You can set EGL preferences for SQL access.

To set EGL SQL preferences, do as follows:

  1. From the main menu, click Window > Preferences. The Preferences window is displayed.
  2. In the Preferences menu, expand EGLand click SQL. The SQL pane is displayed.
  3. In the SQL pane, select the radio button for the option that you want for each of the following values:
    1. Case control rules for naming structure items
    2. Underscore control rules for naming structure items
    3. EGL type to use for SQL date and timestamp in SQL Retrieve
  4. In the SQL pane, select or clear the following check boxes:
    1. Retrieve primary key information from the system catalog
    2. Prompt for SQL user ID and password when needed
  5. To save the changes and remain in the Preferences window, click Apply. To save the changes and exit the window, click OK. To cancel the changes and exit the window, click Cancel.
