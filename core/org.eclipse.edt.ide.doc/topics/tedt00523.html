Editing the EGL build path

This topic tells how to edit the EGL build path. For background information, see "The EGL build path."

To include projects in the EGL build path, do as follows:
  1. In the Project Explorer view, right-click the name of a project that you want to link to other projects and then click Properties. The Properties window opens.
  2. Select the EGL Build Path properties page. A list of all other projects in your workspace is displayed on the Projects page.
  3. Select the check box beside the name of each project that you want to add to the project's build path. For example, if you want to access a project that includes the EGLAR file, specify the project on the Projects page.
  4. If you want to access a stand-alone EGLAR file, specify it on the EGLAR file page:
    • If the file is in a nonbinary project, do as follows:
      1. Click Add EGLAR files. The EGL Selection page is displayed with a list of all the projects that hold EGLAR files.
      2. Expand the project names and click the names of the EGLAR files of interest. You can select multiple names by holding down the Ctrl key during name selection.
      3. Click OK.
    • If the file is in the file system, do as follows:
      1. Click Add External EGLAR files. The EGL Selection page is displayed for you to find one or more files in the file system.
      2. Click the names of the EGLAR files of interest. You can select multiple names by holding down the Ctrl key during name selection.
      3. Click OK.
  5. To place projects and EGLAR files in a different order or to export any of them, click the Order and Export tab and do as follows:
    • To change the position of a project in the build-path order, select the project name and click the Up and Down buttons.
    • To export a project or EGLAR file, select the related check box. To handle all the resources at once, click the Select All or Deselect All button.
  6. Click OK.

If you are generating to Java and want to reference other Java code, you can add a project or JAR file to the Java build path of the project that receives the EGL-generated Java output.

To add a project to the Java build path of a project:

  1. In the Project Explorer view, right-click the name of a project that you want to link to other projects, and then click Properties. The Properties window opens.
  2. Select the Java Build Path properties page.
  3. On the Java Build Path page, click Projects tab.
  4. Click Add,
  5. Select the check boxes next to the projects that you want to add and then click OK.
  6. Click OK again.

To add a referenced project as a JAR file:

  1. Right-click the name of the project that you want to use in your project, and then click Export.
  2. In the Export window, expand Java and click JAR file.
  3. Click Next.
  4. Under Select the resources to export, select the check box next to the project that you want to export.
  5. In the JAR file field, set the location for the JAR file.
  6. Click Finish.
  7. Right-click the name of the project in which you want to use the referenced project, and then click Properties.
  8. In the Properties window, click Java Build Path.
  9. Go to the Libraries tab.
  10. Click Add external JARs.
  11. Select the JAR file that you exported and click Open.
  12. Click OK.
