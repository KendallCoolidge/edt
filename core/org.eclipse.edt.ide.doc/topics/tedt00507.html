Setting EGL general preferences

You can set EGL preferences for EGL Development Tools.

To set the basic EGL preferences, do as follows:

  1. From the main menu, click Window > Preferences. The Preferences window opens.
  2. In the Preferences window, click EGL in the tree.
  3. If you want to change the default EGL folder names, edit the following fields in the EGL folders group:
    • Source folder name
    • Output folder name
  4. To save the changes and remain in the Preferences window, click Apply. To save the changes and exit the window, click OK.
