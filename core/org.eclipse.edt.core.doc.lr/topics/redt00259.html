eglx.xml.binding.annotation package

eglx.xml.binding.annotation defines annotations used in the API for writing data to and from XML strings. This annotation supplements the types in the eglx.xml package.

For an overview of the reading and writing capability, see “Correspondence between an XML string and an EGL variable.”
