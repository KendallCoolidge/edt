Overview of classifiers

The topics subordinate to this one give further details on how to use part types such as Record or Program. The details are grouped as follows:

  • Classifiers for defining EGL data parts

    The classifiers are Data Item, Record, Enumeration, and Class.

  • Classifiers for defining EGL logic parts

    The classifiers are Program, Library, Service, and Handler.

  • Classifiers for defining prototype parts

    The classifiers are Interface, ExternalType, and Delegate.
