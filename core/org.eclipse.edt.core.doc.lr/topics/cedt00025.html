Boxing and unboxing conversions

Assignments to and from compatible fields are straightforward even if you mix reference and value types:

  • When a simple-type reference field receives a value from a value field or literal, a boxing conversion occurs. The source value is assigned to a new memory area, and the address of that area is assigned to the reference field.
  • When a simple-type value field receives a value from a reference field, an unboxing conversion occurs. The value field receives the value to which the reference field refers.
