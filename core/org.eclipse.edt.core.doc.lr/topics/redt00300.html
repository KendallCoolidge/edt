EGLService annotation

The EGLService annotation supports access of EGL REST-RPC services.

Package

eglx.rest

Comments

The fields in the EGLService annotation are as follows:

serviceName string?
Specify the name of the Service type.
requestCharSet string
The character encoding for the request message. The default value is "UTF-8".
requestContentType string?
The content type on the request message.
responseCharSet string?
The character encoding for the response message. This encoding determines how to convert bytes to a string.
