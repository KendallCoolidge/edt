MathLib system library

MathLib includes mathematical and trigonometric functions.

Package

eglx.lang

Functions
Table 1. Functions
System function Purpose
Abs system function MathLib.abs returns the absolute value of a number.
Acos system function MathLib.acos returns the arc cosine (inverse cosine) of a number.
Asin system function MathLib.asin returns the arc sine (inverse sine) of a number.
Atan system function MathLib.atan returns the arc tangent (inverse tangent) of a number.
Atan2 system function MathLib.atan2 computes the principal value of the arc tangent of y/x, using the signs of both arguments to determine the quadrant of the return value.
Ceiling system function MathLib.ceiling returns the smallest integer not less than a specified number.
Cos system function MathLib.cos returns the cosine of an angle that is measured in radians.
Cosh system function MathLib.cosh returns the hyperbolic cosine of a number.
Decimals system function MathLib.decimals returns the number of decimal places in the type of a numeric variable.
Exp system function MathLib.exp returns the exponential of a number, which is the constant e (the base of natural logarithms) raised to the power of that number.
Floor system function MathLib.floor returns the largest integer not greater than a specified number.
Frexp system function MathLib.frexp splits a number into two components: a normalized fraction that ranges from .5 to 1, and an exponent that represents the power of 2.
Ldexp system function MathLib.ldexp takes as input a numeric expression and an exponent that represents a power of 2. The function returns the value of the expression times 2 to the power represented by the exponent.
Log system function MathLib.log returns the natural logarithm of a number.

Log10 system function MathLib.log10 returns the base 10 logarithm of a number.
Max system function MathLib.max returns the greater of two numeric values.

Min system function MathLib.min returns the lesser of two numeric values.

Modf system function MathLib.modf splits a numeric value into two components: an integer and a fraction.
Pow system function MathLib.pow returns a number raised to the power of a second number.
Precision system function MathLib.precision returns the maximum number of digits that are possible in a variable value.
Random system function MathLib.random returns a pseudorandom value between 0.0 and 1.0, inclusive.

Round system function MathLib.round rounds the value of an expression and returns the result.
Sin system function MathLib.sin returns the sine of an angle that is measured in radians.
Sinh system function MathLib.sinh returns the hyperbolic sine of a number.
Sqrt system function MathLib.sqrt returns the positive square root of a number.
Tan system function MathLib.tan returns the tangent of an angle that is measured in radians.
Tanh system function MathLib.tanh returns the hyperbolic tangent of a number.
Comments
The library is implemented as an EGL external type, stereotype NativeType, and not as an EGL library.
