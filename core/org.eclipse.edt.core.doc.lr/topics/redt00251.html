JSONLib.convertToJSON function

JSONLib.convertToJSON copies an EGL record, handler, or dictionary to a JSON string.

Package

eglx.json

Syntax
static function convertToJSON(eglType any const in) returns(string);
eglType
An EGL record, handler, or dictionary.
Example use






Comments



For other details on the conversion, see “Correspondence between
a JSON string and an EGL variable.” 


JSONLib.convertToJSON is
the complement of the JSONLib.convertFromJSON function.
