Cosh system function

MathLib.cosh returns the hyperbolic cosine of a number.

Package

eglx.lang

Syntax
static function cosh(value float in) returns(float);
value
Takes a numeric value.

The returned value is not bounded.

Example use

Here is example code:

result = mathLib.cosh(myVar);
Comments

In the case of Java, EGL uses methods in the Java StrictMath class that are equivalent to the EGL functions. This usage ensures that the runtime behavior is the same for every Java Virtual Machine.
