Statements used for relational database access

The next table lists the statements used to access a relational database.

Statement Purpose
Add To add a row.
Close To close an SQL data source or a result set.
Delete To delete one or more rows.
Execute To run either a prepared SQL statement or a set of one or more non-prepared SQL statements.
ForEach To define a loop that retrieves each row in an SQL result set and operates on the most recently retrieved row by running a block of embedded EGL statements.
Get To retrieve one or more rows.
Open To open a result set.
Prepare To create a prepared statement.
Replace To update a row.
