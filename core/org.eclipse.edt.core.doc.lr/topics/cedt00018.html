Overview of simple types

The topics subordinate to this one give an overview of the simple types, by kind:

  • EGL character types

    The types are BYTES, CHAR, DBCHAR, HEX, MBCHAR, STRING, and UNICODE.

  • EGL date-and-time types

    The types are DATE, INTERVAL, TIME, and TIMESTAMP.

  • EGL large-object types

    The types are BLOB and CLOB.

  • EGL numeric types

    The types are BIGINT, BIN, FLOAT, INT, MONEY, NUM, NUMC, NUMBER, PACF, SMALLFLOAT, and SMALLINT.
