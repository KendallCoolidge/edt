ID annotation

ID marks a field to indicate that the field value represents part of a primary key.

EGL package name

eglx.persistence

Example use
Record Customer {@Table{name = "CUSTOMER_TABLE"}}
   identifier STRING {@ID};
   corporateName STRING;
end
Annotation detail
Record Id type Annotation {
   targets=[FieldMbr]
   }
end
Annotation fields
None.
